# Expedition

AI Music Beat Game for the Snap Games Lensathon.

---

## Code Boundaries & Ownership

| Folder | Owner | Responsibility |
|--------|-------|----------------|
| `/backend` | **Dev 1** | AI music API, beat detection, cloud storage, API endpoints |
| `/lens-studio` | **Dev 2** | Lens Studio project, gameplay, UI integration, PvP sync |
| `/assets` | **Artist** | All visual assets, UI designs, animations, effects |
| `/doc` | **All** | Planning & documentation |

### Rules
- Stay in your folder unless coordinating with the owner
- Shared interfaces (API contracts, asset specs) go in `/doc`
